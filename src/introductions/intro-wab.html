IMPORTANT: exercises and feedback

In order to become familiar with the tool we kindly ask you to do the following exercises:

  • Add comments and tags
  • Create text to text annotations
  • Create triples
  • Annotate a part of an image

After finishing the exercises, please fill in the online user survey. It will give us important hints on how to improve Pundit.

Thank you.
